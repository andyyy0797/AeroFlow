
# AeroFlow: Flight Scheduling & Ticketing System

## Project Overview

This is the Group Project for the course **COMP 2090SEF, HKMU 2025/26 Spring Term**.

Our group has developed an integrated aviation management solution that addresses real-life scheduling challenges through **OOP** and **advanced data structures**.

## Project Structure

The project is divided into two primary tasks:

### Task 1: Main Program

**AeroFlow** is a Python-based flight management system designed to address several problems in real-world avation industry.

**Core Functions**:

*  **Flight Scheduling**: Manage flight routes.

*  **Passenger Services**: User registration and flights booking.

*  **Waitlist**: A system to handle overbooked flights using a **Queue** (FIFO) data structure for fair standby management.

**OOP Implementation**:

* The application uses **Inheritance** (User types, Flight types), **Encapsulation** (Private attributes for seat counts and IDs), and **Polymorphism** (Dynamic pricing strategies).

* The project follows **Modular Programming** with code separated into multiple  functions.
